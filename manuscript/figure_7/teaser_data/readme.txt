EcoDes-DK15 v1.1.0 teaser / sample data set - "Husby Klit" - December 2021
Jakob J. Assmann, Jesper E. Moeslund, Urs A. Treier and Signe Normand
Contact: j.assmann@bio.au.dk

Subsample of the EcoDes-DK15 data set for demonstration purposes. This subset 
contains all terrain and vegetation variables, as well as all auxiliary data for 
the nine 1 km x 1 km tiles that cover the area of the Husby Klit plantation shown 
in Figure 7 of the manuscript. 

This data is freely available via a Creative Commons by Attribution 4.0 license.

For the full dataset and documentation, and information on how to cite this data 
please see:

https://doi.org/10.5281/zenodo.4756556
https://github.com/jakobjassmann/ecodes-dk-lidar
